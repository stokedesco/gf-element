This plugin allows you to easily integrate Gravity Forms into your Elementor layouts with a customizable widget.

== Description ==

**Gravity Forms Widget for Elementor** is a simple but powerful tool that allows you to add and customize Gravity Forms within Elementor page layouts. No need to mess with shortcodes! Just drag and drop the widget, select your form, and customize it directly from Elementor.

With this plugin, you can:

- Select any Gravity Form to display.
- Customize the form's appearance within Elementor.
- Seamlessly integrate Gravity Forms functionality into your Elementor designs.

**Key Features:**

- Full compatibility with Gravity Forms.
- Display any Gravity Form with a simple drag-and-drop interface.
- Customize your forms using Elementor's powerful editor.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/gravity-forms-elementor-widget/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to your Elementor page editor, search for "Gravity Forms" in the Elementor widget panel, and drag the Gravity Forms widget into your layout.
4. Select the Gravity Form you want to display and customize the settings as desired.

== Frequently Asked Questions ==

= Do I need Gravity Forms installed to use this plugin? =
Yes, Gravity Forms must be installed and activated for this widget to function.

= Does this plugin support all Gravity Forms features? =
This plugin supports the basic functionality of displaying and customizing Gravity Forms. Advanced Gravity Forms features should still work, but for certain specialized cases, you may need additional customization.
